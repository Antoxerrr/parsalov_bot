ΔU = ΔQ - pΔV,
приращения здесь и далее считаем малыми и полагаем, что внутренняя энергия
зависит только от температуры (идеальный газ).
При ΔV = 0 имеем ΔU = ΔQ = СVΔT, откуда ΔU = СVΔT и
ΔQ = CV ΔT + pΔV.

Из уравнения состояния идеального газа для одного моля находим:
RΔT = pΔV + VΔP.
Подставляем ΔT в предыдущее уравнение:
ΔQ = (CV/R)*(pΔV + VΔP) + pΔV.
Находим:
ΔA/ΔQ = (pΔV)/ΔQ = 1/((CV/R)*(1 + (V/ΔV)*(ΔP/P)) + 1)
Здесь CV - молярная теплоемкость.

Подставим численные значения:
СV/R = (cv*μ)/R = 745*(28E-3)/8.41 = 2.51,
здесь СV - молярная теплоемкость, cv - удельная теплоемкость, данная в условии.
В результате получим (учитывая, что ΔP < 0):
ΔA/ΔQ = 1/(2.51*(1-1/2)+1) = 0.443
